T1 math

题目描述

hzy9819最近对数学产生了兴趣，尤其是一些奇奇怪怪的数列。

有一天他心血来潮想到了一个题目：

求$$\sum_{i=1}^{n}{\sum_{j=1}^{i}{S_{i,j} \times F_{j} \times P_{j}}}$$

其中$F_i$表示长度为$i$的错位排列的个数

$P_i$表示长度为$i$的圆周排列的个数

$S_{i,j}=j\times S_{i-1,j}+S_{i-1,j-1},S_{i,0}=[i=0] $

答案对998244353取模

输入格式

一行一个正整数$n$

输出格式

一行一个整数表示答案

数据范围

10%的数据满足$n \le 10$

30%的数据满足$n \le 1000$

50%的数据满足$n \le 20000$

100%的数据满足$n \le 100000$

题解

错排公式：$f_n=(n-1)(f_{n-1}+f_{n-2})$。
圆周排公式：$f_n=(n-1)!$。
S不好处理考虑S的组合意义。
首先通过观察我们发现$S_{i,j}$实际上足将i个有标号的球放入j个无标号的盒子的方案数(不允许有空盒)那么$S_{i,j}\times j!$就足将i个有标号的球放入j个有标号的盒子的方案数(不允许有空盒)再进一步考虑：如果允许有空盒那么方案数就是$j^i$，那么对上式容斥后
可以得到:
$$S_{n,m} \times m!=\sum_{i=0}^mC_m^i(-1)^i(m-i)^n$$
代入原式可得：

$$\sum_{i=1}^{n}{\sum_{j=1}^{i}{S_{i,j} \times P_{j} \times F_{j}}}$$

$$\sum_{i=0}^{n}{\sum_{j=1}^{n}{S_{i,j} \times P_{j} \times F_{j}}}$$

$$=\sum_{i=0}^{n}{\sum_{j=1}^{n}}{\frac{F_{j}}{j}{\sum_{k=0}^{j}{\binom{j}{k}(-1)^k(j-k)^i}}}$$

$$=\sum_{j=1}^{n}{\frac{F_{j}}{j}{\sum_{k=0}^{j}{\binom{j}{k}(-1)^k{\sum_{i=0}^{n}{(j-k)^i}}}}}$$

$$=\sum_{j=1}^{n}{\frac{F_{j}}{j}{\sum_{k=0}^{j}{\frac{j!}{k!(j-k)!}(-1)^k{\frac{(j-k)^n-1}{(j-k)-1}}}}}$$

$$=\sum_{j=1}^{n}{F_i\times (j-1)!{\sum_{k=0}^{j}{\frac{(-1)^k}{k!}\times{\frac{(j-k)^n-1}{(j-k)-1}\frac{1}{(j-k)!}}}}}$$

上式已经是一个很明显的卷积式子了，直接NTT即可

时间复杂度$O(nlogn)$

T2 sequence

题目描述

小R最近对一些奇怪的序列展开了研究。

他发现一个序列是${a_{n}}$“优秀的”当且仅当$\prod_{i=1}^{n-1}{\binom{a_{i}}{a_{i+1}}}$为奇数。

现在小R手上有一个序列，为了进一步的研究，小R想让你帮他统计出“优秀的”子序列有多少个。

答案对998244353取模。

提示:$\binom{n}{k} = \frac{n!}{k!(n-k)!}$

输入格式

第一行一个正整数$n$表示序列长度

第二行$n$个数描述序列$a$

输出格式

一行一个整数表示答案

数据范围

20%的数据满足$n \le 10$

另20%的数据满足$n,Max{a_{i}} \le 1000$

另10%的数据满足数据随机

100%的数据满足$n \le 10^5$，$Max{a_{i}} < 131072$

题解

乘积为奇数等价于原式中的每一项都为奇数，由lucas定理可知$\binom{n}{k}$为奇数当且仅当$n|k=n$。

考虑暴力dp如何优化，由于位运算的独立性，我们可以把每个$a_{i}$拆成$b_{i} \times 2^8 + c_{i}$的形式，其中$b_{i},c_{i} \le 2^9$。计$S_{i,j}$ 表示b=i且c是j的子集的dp值之和。那么转移时就只需要枚举$b_{i}$的子集，然后对包含$c_{i}$的集合的S进行更新。

时间复杂度$O(2^9n)$

空间复杂度$O(n)$

T3 phi

题目描述

cab对欧拉函数$\varphi(n)$特别感兴趣，决定造个大新闻。

现在cab手上有一个序列${a_{n}}$，他想知道$\varphi(\prod_{i=l}^{r}{a_{i}})$对$998244353$取模后的值，其中$l,r$是每次询问给定的两个参数。

由于你觉得cab的这个题太excited了，你决定帮他解决这个题。

输入格式

第一行一个正整数$n$表示序列长度

第二行$n$个正整数描述序列$a$

第三行一个正整数$q$表示询问数

接下来$q$行每行描述一个询问

输出格式

$q$行每行一个整数表示对应询问的答案

数据范围

10%的数据满足$n,q \le 10$

20%的数据满足$n,q \le 1000$

60%的数据满足$n,q \le 30000$

100%的数据满足$n,q \le 200000,Max{a_{i}} \le 10^6$

题解

考虑如何计算$\phi(n)$。设n的质因子分解为$\prod p_i^{a_i}$ 则$\phi(n)=\prod (p_i-1)p_i^{a_i-1}$。很容易想到一个莫队算法，在现在的答案区间内加减一个数只要考虑它的所以质因子对答案的贡献即可。每个数最多只有7个不同的质因子，复杂度为$O(7n\sqrt n)$ 只有60分。

对于这类问题还有一个常见思路是将询问按右端点排序，然后用线段树维护对于某个右端点，左端点为某个值时的答案，然后考虑右端点的移动对答案的影响。想想这题似乎也适用。当加入一个数时考虑它的所有质因子对答案的贡献，记它第i个质因子为$a_i$,个数为$b_i$,上一次出现的位置为$c_i$，那么对于左端点为$c_i+1\to i$ 的答案应乘上$(a_i-1)\times a_i^{b_i-1}$ 因为在此之前这个质因子从未出现过。对于左端为$1\to c_i$ 的答案应乘上$a_i^{b_i}$。于是我们可以用线段树维护。稍微转化一下也可以用树状数组维护。复杂度$O(7nlogn)$ 。
