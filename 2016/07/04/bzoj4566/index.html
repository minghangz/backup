题目链接

bzoj4566

题目描述

Description

给定两个字符串，求出在两个字符串中各取出一个子串使得这两个子串相同的方案数。两个方案不同当且仅当这两
个子串中有一个位置不同。

Input

两行，两个字符串s1，s2，长度分别为n1，n2。1 <=n1, n2<= 200000，字符串中只有小写字母

Output

输出一个整数表示答案

Sample Input

aabb
bbaa

Sample Output

10

题解

将两个串一起建一个广义后缀自动机，每个节点记录一下在两个串中的出现次数，最后拓扑排序一下更新这个次数。由于后缀自动机的性质，只需按step为关键字排序即可。出现次数求出来后随便算算即可。
